# Loggex

Logging utilities for Elixir.

## Installation

The package can be installed
by adding `loggex` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:loggex, "~> 0.0.1"}
  ]
end
```

Documentation can be found at [https://hexdocs.pm/loggex](https://hexdocs.pm/loggex).
